# Older Versions

This document provides information about earlier versions of the MuMo project that preceded the final Version 3 hardware.

## Version 1

Version 1 of the nodes was documented in the following [repository](https://github.com/MoMu-Antwerp/MuMo-v1).

### Gateways
Version 1 included gateways that were built using a Raspberry Pi with a camera module. All details are documented in the same [Version 1 repository](https://github.com/MoMu-Antwerp/MuMo-v1).

## Version 2

Compared to Version 1 which was built with off-the-shelf components, Version 2 used a custom designed PCB board. 

- **[Hardware repository](https://github.com/Strooom/MuMo-V2-Node-PCB)**
- **[Software repository](https://github.com/Strooom/MuMo-v2-Node-SW)**
